## Create Treasure/Clue/Food tokens
- Strike It Rich: Create a Treasure token. (It's an artifact with "{T}, Sacrifice this token: Add one mana of any color.")
- Ruthless Knave: {2}{B}, Sacrifice a creature: Create two Treasure tokens. (They're artifacts with "{T}, Sacrifice this token: Add one mana of any color.")
- Sokka's Sword Training: Create a Clue token. (It's an artifact with "{2}, Sacrifice this token: Draw a card.")
